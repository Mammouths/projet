## Description

Splatoon-ish, consiste en une toile en Spandex sur laquelle est projeté un environnement dans lequel les interacteurs peuvent répandre de la couleur.
L'objectif est de couvrir la plus grande surface possible dans un temps donné, tout en évitant de laisser les autres intéracteurs prendre notre espace.
Avant que l'installation détecte des joueurs (personnes proches du projet), elle affiche des petites *boules d'énergie* qui représente les personnes qui passent devant la toile.
Celles-ci laissent une trainée quand elles bougent et se dissipent avec le temps. Cet effet est présent pour inciter les interacteurs à s'approcher de l'installation.
