## Description



Arbre en Face est une installation interactive où l’humain et le végétal fusionnent : chaque feuille des arbres arbore le visage des participants. Dans cet écosystème numérique, le public doit appuyer dans la toile afin d'arroser les arbres et les protéger d'un soleil mobile qui brûle tout sur son passage. La survie de l’écosystème repose sur une alliance fragile : il faut coopérer pour contrer la sécheresse, mais veiller à ne pas arroser tous en même temps sous peine de provoquer une inondation destructrice.
